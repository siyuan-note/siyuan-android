SiYuan — это система управления личными знаниями, ориентированная на конфиденциальность, поддерживающая детализированные ссылки на уровне блоков и WYSIWYG-редактор Markdown.

<h3>Возможности</h3>
Большинство функций бесплатны, даже для коммерческого использования.
<ul>
<li>Блок содержимого</li>
<ul>
<li>Ссылки на уровне блока и двусторонние ссылки</li>
<li>Пользовательские атрибуты</li>
<li>Встраивание SQL-запроса</li>
<li>Протокол <code>siyuan://</code></li>
</ul>
</li>
<li>Редактор</li>
<ul>
<li>Стиль блока</li>
<li>WYSIWYG-редактор Markdown</li>
<li>Списочная структура</li>
<li>Масштабирование блока</li>
<li>Горизонтальная компоновка блока</li>
<li>Редактирование больших документов объёмом в миллион слов</li>
<li>Математические формулы, диаграммы, блок-схемы, диаграммы Ганта, диаграммы синхронизации и т.д.</li>
<li>Веб-клиппинг</li>
<li>Ссылка аннотации PDF</li>
</ul>
</li>
<li>Экспорт
<ul>
<li>Ссылка на блок и встраивание</li>
<li>Стандартный Markdown с ресурсами</li>
<li>PDF, Word и HTML</li>
<li>Копировать в WeChat MP, Zhihu и Yuque</li>
</ul>
</li>
<li>База данных
<ul>
<li>Табличный вид</li>
</ul>
</li>
<li>Интервальное повторение</li>
<li>Многовкладочный режим, перетаскивание для разделения экрана</li>
<li>Фрагмент шаблона</li>
<li>Фрагмент JavaScript/CSS</li>
<li>Приложение для Android/iOS</li>
<li>Развертывание в Docker</li>
<li>API</li>
<li>Маркетплейс сообщества</li>
</ul>

Некоторые функции доступны только платным участникам. Для получения более подробной информации см. Pricing.
